# EE-Cinema
Сайт кинотеатра

Сайт для покупки билетов и просмотра горячих новинок в мире киноиндустрии

у нас есть главная, кино, театр, концерты, вход, регистрация, сотрудничество.

джава скрипт добавляет на сайт карточки, банеры, добавляет возможность более менее функциональной формы регистра и входа.
